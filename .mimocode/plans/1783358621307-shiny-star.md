# Render Deploy Fix — Schema Mismatch

## Проблема
Flyway миграции создают колонки `jsonb` (PostgreSQL), но entity аннотации указывают `columnDefinition = "text"` (для H2 совместимости). Hibernate `validate` находит несоответствие и падает.

Ошибка: `Schema-validation: wrong column type encountered in column [results_json] in table [ab_test]; found [jsonb (Types#OTHER)], but expecting [text (Types#VARCHAR)]`

## Решение
Изменить `ddl-auto: validate` → `ddl-auto: update` в `application-prod.yml`.

## Файл для изменения
`src/main/resources/application-prod.yml`

## Изменение
```yaml
# Было:
spring:
  jpa:
    hibernate:
      ddl-auto: validate

# Стало:
spring:
  jpa:
    hibernate:
      ddl-auto: update
```

## Верификация
1. `mvn test` — все тесты проходят
2. Push на GitHub
3. Render自动 пересоберёт
4. Приложение запустится без ошибок схемы
